# Simple Countdown Game

A simple countdown game made with **Python**.  
Play three different countdown games: timer, guessing game, and reaction challenge.

## How to run

1. Make sure you have Python installed (Python 3.6 or higher).
2. Open a terminal in this folder.
3. Run the game:

   ```bash
   python countdown_game.py
   ```

## Games

1. **Simple Countdown Timer** - Enter a number and watch it count down to zero.
2. **Guess the Countdown** - Try to guess the starting number before the countdown begins.
3. **Stop at Zero Challenge** - Test your reaction time by pressing Enter when the countdown hits zero.

## Requirements

- Python 3.6 or higher
- No external libraries needed (uses only built-in `time` and `random` modules)

## Files

- `countdown_game.py` - The main game file with all three game modes.
